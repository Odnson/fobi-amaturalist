# Auth Images

Folder ini berisi gambar background untuk halaman Login dan Register.

## File yang Diperlukan

1. **login-bg.jpg** - Gambar background untuk form Login
2. **register-bg.jpg** - Gambar background untuk form Register

## Spesifikasi Gambar

- **Format**: JPG, PNG, atau WebP
- **Ukuran Rekomendasi**: 800x600 pixels atau lebih besar
- **Orientasi**: Portrait atau Square
- **Rasio Aspek**: 4:3 atau 3:4 (untuk hasil terbaik)
- **Ukuran File**: Maksimal 2MB untuk performa optimal

## Cara Mengganti Gambar

1. Siapkan gambar yang ingin digunakan
2. Rename gambar sesuai dengan nama file yang diperlukan:
   - `login-bg.jpg` untuk halaman login
   - `register-bg.jpg` untuk halaman register
3. Upload/copy file ke folder ini (`public/auth-images/`)
4. Refresh halaman untuk melihat perubahan

## Catatan

- Jika gambar tidak ditemukan, sistem akan menggunakan gradient warna sebagai fallback
- Gambar akan di-cover (object-fit: cover) untuk mengisi seluruh area
- Ada overlay gelap otomatis untuk meningkatkan keterbacaan
- Pastikan gambar memiliki kualitas yang baik dan relevan dengan tema biodiversity/nature

## Contoh Gambar yang Cocok

- Foto satwa liar (burung, kupu-kupu, mamalia, dll)
- Foto flora/tanaman
- Foto habitat alami
- Foto landscape alam Indonesia
- Foto aktivitas citizen science

## Tips

- Gunakan gambar dengan subjek yang jelas dan menarik
- Hindari gambar yang terlalu ramai atau membingungkan
- Pastikan gambar memiliki kontras yang baik
- Pertimbangkan untuk menggunakan gambar yang berbeda untuk login dan register agar lebih menarik
